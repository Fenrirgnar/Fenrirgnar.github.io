Навыки

Что я использую в работе

HTML

Именно он создает каркас вашего сайта или приложения, а пятая версия позволит мне создавать более SEO-оптимизированную структуру вашего продуктa

CSS3

Этот язык стилей позволяет мне создавать абсолютно любой внешний вид вашего сайта или приложения. Все ограничивается только вашей фантазией!

Java Script

Этот язык программирования позволяет оживить все что угодно: слайдеры, окна, подсказки, вкладки, получение данных от сервера и многое другое. Этот аспект я только начинаю осваивать.

Angular

Библиотека Angular позволит ускорить разработку, планирую освоить её как буду уверенно знать нативный JS

React

Эта библиотека позволяет создавать web-приложения. Также планирую освоить со временем

Node.js

Эта платформа позволяет создавать бэкенд для вашего продукта - “мозги”, которые будут выполнять действия, которые пользователь не видит. Считаю что это стезя бэкенд разработчиков но для общего понимания планирую изучить работу платформы

Mongo.db

Это нереляционная база данных, которая будет хранить данные вашего сайта или приложения. Отдаленно знаком,боже храни ютуб.
